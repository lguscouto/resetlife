# ResetLife `0.3.2` — Arquitetura de informação e navegação

**Data da validação:** 12/07/2026  
**Status:** build debug gerado, testes unitários, pipeline instrumentado e fluxo visual validado  
**Version code:** `5`  
**Version name:** `0.3.2`

## Resumo

A versão `0.3.2` melhora a arquitetura de informação do aplicativo sem adicionar um novo módulo de negócio. O usuário passa a ter destinos de navegação nomeados, rótulos mais claros, descrições acessíveis, títulos de tela mais objetivos e retorno previsível pelo botão do Android.

## O que foi entregue

- enumeração explícita de destinos `Today` e `Organize`;
- substituição dos índices mágicos `0` e `1` na seleção de aba;
- componente reutilizável `ResetLifeNavigationBar`;
- rótulos visíveis e descrições de acessibilidade para cada destino;
- indicador de destino selecionado com os tokens do tema;
- preservação da aba selecionada com `rememberSaveable`;
- preservação de ViewModels fora da seleção de destino;
- botão voltar do Android retornando de Organizar para Hoje antes de fechar a Activity;
- TopAppBar da tela Hoje com o título Hoje;
- título principal da tela Hoje ajustado para “Prioridades de hoje”;
- TopAppBar da tela Organizar padronizada com o fundo do tema;
- hierarquia de Projetos, Nova tarefa e Pendências preservada e reforçada;
- strings de navegação e acessibilidade mantidas em recursos PT-BR;
- nenhuma alteração no Room, entidades, repositórios ou regras de negócio.

## Limitações desta release

- ainda não há testes instrumentados automatizados específicos para interação de navegação;
- a validação visual foi feita por hierarquia de acessibilidade e interação no emulador, não por comparação de screenshots;
- o APK gerado é debug instalável, não um artefato release assinado para publicação;
- o próximo incremento `0.3.3` deve aprofundar a jornada Hoje conforme o roadmap UX/UI.
