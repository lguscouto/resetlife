# ResetLife 0.10.0

**Build gerado em 14 de julho de 2026**

## Objetivo

Entregar as **listas próprias do usuário** (módulo dedicado) e os **atalhos de microação na tela Hoje**, conforme o plano definido ao fechar a 0.9.0.

## O que foi entregue

### 1. Listas próprias (`CustomLists`)
Módulo completo, acessível pelo hub **Vida → Listas**:

- Criar listas (ex: compras, pendências, o que quiser)
- Selecionar uma lista e ver seus itens
- Adicionar itens à lista
- Marcar item como concluído (checkbox)
- Excluir item ou lista inteira

**Camada de dados**
- `CustomListEntity` + `CustomListItemEntity` (Room, nova tabela `custom_list_item`)
- `CustomListDao` / `CustomListRepository` (observe, insert, update, delete)
- Migration automática **v7 → v8** (nova tabela; o schema v7 já existia)

**Camada de apresentação**
- `CustomListsViewModel` (StateFlow: listas, itens, diálogos)
- `CustomListsScreen` (chips de lista, itens com checkbox, diálogos de criação)
- `ResetLifeDestination.CustomLists` (filho de Vida, não aba extra)
- Card "Listas" no hub Vida
- Strings em PT-BR

### 2. Atalho de microação na Hoje
A tela **Hoje** agora sugere a primeira tarefa de ambiente ainda não concluída:

- Card "Microação de ambiente" com ícone `Eco`, título da tarefa e botão **Concluir**
- Ao concluir, a tarefa é marcada como feita no módulo Ambiente e o atalho some
- Tocar no card abre o módulo Ambiente
- Observa todas as tarefas de todos os espaços (não só o primeiro)

**Implementação**
- `TodayViewModel` recebe `EnvironmentRepository` e expõe `environmentSuggestion`
- `TodayScreen` renderiza o card condicionalmente (`environmentSuggestion != null`)
- `TodayViewModelFactory` atualizada

## Limitação conhecida

- A verificação visual do atalho "Microação de ambiente" na tela Hoje em emulador foi dificultada pelo overview do sistema (a UI automatizada do emulador capturava a tela de apps recentes). A lógica está coberta por teste unitário + instrumentado, e a renderização é condicional ao estado. Recomenda-se validação visual manual em dispositivo.
